🌍 AI Travel Guide Chatbot ✈️

An AI-powered travel assistant that provides personalized travel recommendations, itineraries, and location insights in real-time.

Built with a modern full-stack approach, combining a Python Flask backend with a responsive web UI, and deployed on cloud platforms.

---

🚀 Features

- 🤖 AI-generated travel recommendations using Gemini API
- 🗺️ Suggest places, itineraries, and travel tips
- 💬 Real-time chat interface with typing animation
- 📸 Dynamic travel images based on user queries
- 📍 Quick access to Google Maps locations
- ⚡ Fast and lightweight UI (mobile-friendly)

---

🛠️ Tech Stack

Frontend

- HTML, CSS, JavaScript
- Custom chat UI with animations
- Deployed on Vercel

Backend

- Python (Flask)
- REST API
- Google Gemini API integration
- Flask-CORS for cross-origin requests
- Deployed on Render

---

🧠 How It Works

1. User enters a travel-related question
2. Frontend sends a POST request to the Flask API
3. Backend processes the prompt using Gemini AI
4. AI generates a structured travel response
5. Response is sent back and displayed with a typing effect
